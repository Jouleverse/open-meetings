# Web2与Web3的进阶转换
- Web2平台问题：芳芳提到Web2电商平台存在虚假图片、退货麻烦等问题。
- Web3改变现状：芳芳受教链的启发，探讨Web3是否能改变Web2平台的现状。
- 芳芳分享了教链对Web3的愿景，认为Web3可以通过去中心化的方式改进现有的电商服务。

# 去中心化的现实应用
- 社会框架的中心化：如行提到，目前的社会仍然是中心化构建的框架，Web3的出现针对一些痛点，但不能完全颠覆现有体系。
- Web3的利基市场：如行认为，Web3需要找到自己的利基市场，与现实世界结合才能找到力量并生存下去。
- 去中心化的必要条件：如行指出，去中心化是必要的，但不是充分条件，还需演化出适合现实世界的形态才能真正生存和发展。

# Web3的核心理念与发展
- Web3的核心：如行认为Web3的核心是让人在网络上真正拥有一串数字，并通过去中心化的方式实现共同决定。
- 观念与现实的碰撞：如行强调观念需要与现实碰撞，只有在这个过程中才能找到Web3真正的落脚点和生命力。
- 盈利与生命力：如行提到，Web3的发展需要注重盈利，这样才能够长久生存下去，并吸引更多的人才。

# Web3的现实挑战与应对
- Web3的有机成长：如行认为Web3的发展需要一个有机成长的过程，要适应现实世界的客观环境，而不是仅凭观念。
- 法律法规的认知差异：如行指出，不同国家和地区对Web3的法律法规和认知存在差异，需要结合具体情况来应对。
- Web3的外界融合：如行强调Web3需要与现实世界融合，找到生长空间，在规则框架下不断成长，吸引更多人才。
